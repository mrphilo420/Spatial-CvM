BASIC DEFINITIONS

1. SPATIAL LOCATIONS AND DISTANCES
-----------------------------------

Definition: Loc := ℝ × ℝ
  Spatial locations as points in [0,1]² (or ℝ²)

Definition: loc_norm (s : Loc) : ℝ := √(s₁² + s₂²)
  Euclidean norm in ℝ²

Definition: loc_dist (s₁ s₂ : Loc) : ℝ := loc_norm (s₁ - s₂)
  Euclidean distance metric: d(s₁, s₂) = √((s₁ₓ - s₂ₓ)² + (s₁ᵧ - s₂ᵧ)²)

Lemma loc_dist_nonneg: 0 ≤ loc_dist s₁ s₂ [PROVED]
Lemma loc_dist_comm:   loc_dist s₁ s₂ = loc_dist s₂ s₁ [PROVED]
Lemma loc_dist_triangle: loc_dist s₁ s₃ ≤ loc_dist s₁ s₂ + loc_dist s₂ s₃ [PROVED]
  Triangle inequality via Minkowski inequality for ℓ² norm

2. KERNEL FUNCTIONS
-------------------

Structure IsKernel (K : ℝ → ℝ) :=
  ├─ symmetric: K(x) = K(-x)
  ├─ bounded: ∃B > 0, ∀x, |K(x)| ≤ B
  ├─ supported: ∃r > 0, ∀x, |x| > r ⟹ K(x) = 0
  └─ lipschitz: ∃L > 0, ∀x y, |K(x) - K(y)| ≤ L|x - y|

Definition: kernel_scaled K h := λx. (1/h²) K(x/h)
  Bandwidth-scaled kernel

Lemma kernel_scaled_integrable: |kernel_scaled K h x| ≤ C [PROVED]
  Scaled kernel remains bounded

3. RANDOM FIELDS
----------------

Axiom IsStationary (X : Loc → ℝ) : Prop
  Spatial random field is stationary under translation

Axiom IsIsotropic (X : Loc → ℝ) : Prop
  Spatial random field is isotropic under rotation

Axiom AlphaMixing (α : ℝ → ℝ) : Prop
  α-mixing (strong mixing) coefficients satisfy decay conditions

Definition: rotate_loc (s : Loc) (θ : ℝ) : Loc
  Rotation by angle θ in ℝ²

Axiom gamma (X : SpatialField) (h : ℝ) : ℝ
  Covariance function: γ(h) = Cov(X(0), X(h))

Axiom alpha_mixing_covariance_decay:
  IsStationary X ∧ AlphaMixing α ⟹
  ∃δ > 0, ∀h > 0, |γ(h)| ≤ 4·α(h)^δ
  Covariance decays under α-mixing via Davydov inequality

LEMMA 1: ASYMPTOTIC COVARIANCE
================================================================================

STATEMENT:
Under fixed bandwidth h and α-mixing assumption:

  ∃Γ : ℝ → ℝ,
    Γ(0) > 0                                          [Non-consistency]
    ∧ Γ is continuous                                 [Continuity]

where Γ is the asymptotic covariance operator of the kernel-smoothed
empirical process.

FORMALIZATION in Lean:

axiom asymptotic_covariance (K : ℝ → ℝ) (h : ℝ) (hh : h > 0)
    (hK : IsKernel K)
    (α : ℝ → ℝ) (h_mix : AlphaMixing α) :
    ∃ Γ : ℝ → ℝ,
    (Γ 0 > 0) ∧
    (∀ ε > 0, ∃ δ > 0, ∀ s₁ s₂, |s₁ - s₂| < δ → |Γ s₁ - Γ s₂| < ε)

KEY INSIGHT:
Unlike classical empirical CDF theory, at FIXED bandwidth h, the covariance
does NOT vanish. This is the fundamental source of non-consistency that
characterizes the spatial CvM test.

SUPPORTING DEFINITIONS:

Definition: gamma_kernel (K : ℝ → ℝ) (h : ℝ) (u : ℝ) : ℝ
  Kernel-weighted indicator covariance: γₖ,ₕ(u) = ∫ Kₕ(v)Kₕ(v-u) dv

Definition: Gamma_operator (K : ℝ → ℝ) (h : ℝ) (s₁ s₂ : ℝ) : ℝ
  Asymptotic covariance operator: Γ(s₁,s₂) = ∫ Kₕ(u-s₁)Kₕ(u-s₂) du

Lemma gamma_kernel_pos_at_zero:
  ∫ Kₕ(v)² dv > 0                                   [PROVED via axiom]

THEOREM 1: WEAK CONVERGENCE IN ℓ∞([0,1]²)
================================================================================

STATEMENT:
Under fixed-domain infill asymptotics with α-mixing:

  ∃Z (Gaussian process),
    Xₙ ⟹ Z in D([0,1]²)
    ∧ Cov(Z) is given by Γ from Lemma 1

where Xₙ is the kernel-smoothed empirical process at n sample sites.

FORMALIZATION in Lean:

Axiom IsGaussian (Z : ℝ → ℝ) : Prop
  Property: Z is a Gaussian process

Axiom IsTight (Xₙ : ℕ → ℝ → ℝ) : Prop
  Tightness condition for process sequences

Axiom prokhorov_theorem:
  (finite-dimensional convergence) ∧ (tightness) ⟹
  weak convergence in D([0,1]²)

Axiom gaussian_process_exists:
  ∀μ, Γ symmetric and positive semi-definite ⟹
  ∃Z Gaussian with mean μ and covariance Γ

axiom weak_convergence (K : ℝ → ℝ) (h : ℝ) (hh : h > 0)
    (α : ℝ → ℝ) (h_mix : AlphaMixing α) (δ : ℝ) (hδ : δ > 0) :
    ∃ Z : ℝ → ℝ,
    IsGaussian Z ∧
    (∀ t₁ t₂, Gamma_operator K h hh t₁ t₂ = Gamma_operator K h hh t₂ t₁)

KEY THEOREM COMPONENTS:
1. Finite-dimensional distributions converge to Gaussian (from Lemma 1)
2. Tightness holds via moment bounds (from Lemma 1 + α-mixing)
3. Prokhorov's theorem ⟹ weak convergence
4. Limit covariance is given by Γ from Lemma 1

THEOREM 2: ASYMPTOTIC NULL DISTRIBUTION
================================================================================

STATEMENT:
Under H₀ (spatial homogeneity) with α-mixing:

  T_n ⟹ Σₘ λₘ* χ²ₘ(K-1)

where:
  • T_n is the kernel-smoothed CvM test statistic
  • λₘ* are eigenvalues of the contrast covariance operator
  • χ²ₘ(K-1) are independent χ²(K-1) random variables
  • The sum is a weighted chi-square distribution

FORMALIZATION in Lean:

Definition: weighted_chisq (λ : ℕ → ℝ) (k : ℕ) : ℝ
  Weighted chi-square: Σₘ λₘ * χ²ₘ(k)

axiom asymptotic_null (K : ℕ) (hK : K ≥ 2)
    (h : ℝ) (hh : h > 0)
    (α : ℝ → ℝ) (h_mix : AlphaMixing α) (δ : ℝ) (hδ : δ > 0) :
    ∃ (lam : ℕ → ℝ) (limit_dist : ℝ),
    (∀ m, lam m ≥ 0) ∧
    (limit_dist = weighted_chisq lam (K - 1))

KEY THEOREM COMPONENTS:
1. Under H₀, spatial pattern reduces to weighted contour problem
2. Spectral decomposition of contrast covariance via Mercer's theorem
3. Delta method applied to smooth test functional
4. Result: asymptotic null distribution is weighted χ²

MERCER DECOMPOSITION:
The contrast covariance operator admits Mercer decomposition:

  Γ(s,t) = Σₘ₌₁^∞ λₘ φₘ(s) φₘ(t)

where:
  • λₘ ≥ 0 are eigenvalues (summable: Σλₘ < ∞)
  • φₘ are orthonormal eigenfunctions: ∫φₘ(s)φₙ(s)ds = δₘₙ
  • The series converges uniformly on [0,1]²

THEOREM 3: MULTIVARIATE EXTENSION VIA COPULAS
================================================================================

STATEMENT:
For multivariate spatial data X₁, ..., Xₚ with p ≥ 2 components:

  Tₙ⁽ᵖ⁾ ⟹ Σₘ λₘ⁽ᵖ⁾,* χ²ₘ(K-1)

where λₘ⁽ᵖ⁾,* are eigenvalues of the p-dimensional contrast covariance
computed via copula transformation.

FORMALIZATION in Lean:

Definition: copula_test_statistic (p K h : parameters) (n : ℕ) : ℝ
  Multivariate CvM test combining all p components via copulas

Axiom copula_mercer_decomposition (p K : ℕ) (hK : K ≥ 2) (hp : p ≥ 2) :
    ∃ (λ : ℕ → ℝ) (φ : ℕ → ℝ → ℝ),
    (∀ m, λ m ≥ 0) ∧
    (∀ m n, ∫ x, φ m x * φ n x = δₘₙ) ∧
    (∀ s t, 1 = Σₘ λ m * φ m s * φ m t)

Axiom copula_trace_class (p K : ℕ) (hK : K ≥ 2) (hp : p ≥ 2) :
    ∃ (λ : ℕ → ℝ), (∀ m, λ m ≥ 0) ∧ (Σₘ λ m < ∞)

Axiom copula_weak_convergence (K p : ℕ) ... :
    Tendsto (fun n => copula_test_statistic p K h n)
            Filter.atTop
            (𝓝 (Σₘ λ m * ChiSquare (K - 1)))

theorem multivariate_limit (K p : ℕ) (hK : K ≥ 2) (hp : p ≥ 2) ... :
    ∃ λ : ℕ → ℝ,
    ∃ limit_dist : ℝ,
    (∀ m, λ m ≥ 0) ∧
    (Σₘ λ m < ∞) ∧
    (Tendsto ... (𝓝 limit_dist)) ∧
    (limit_dist = Σₘ λ m * ChiSquare (K - 1))

PROOF STRATEGY:
1. Copula transformation to uniform margins
2. Apply Theorem 1 componentwise to copula process
3. Mercer decomposition of copula covariance operator
4. Functional delta method applied to copula contrast
5. Result: same weighted χ² structure as Theorem 2

Definition: IsLittleO (f g : ℕ → ℝ) : Prop :=
  ∀ε > 0, ∃N, ∀n ≥ N, |f(n)| ≤ ε·|g(n)|
  "f is little-o of g": f(n)/g(n) → 0

Definition: IsBigO (f g : ℕ → ℝ) : Prop :=
  ∃C > 0, ∀n, |f(n)| ≤ C·|g(n)|
  "f is big-O of g": |f(n)| bounded by |g(n)| up to constant

Lemma littleO_of_tendsto_zero [PROVED]:
  IsLittleO f g ↔ Tendsto (fun n => f(n)/g(n)) atTop (𝓝 0)
  
  Proof establishes the fundamental equivalence between:
  - ε-δ definition of little-o via ratio bounds
  - Metric space definition of convergence to zero

================================================================================
KEY MATHEMATICAL INSIGHTS FORMALIZED
================================================================================

1. NON-CONSISTENCY AT FIXED BANDWIDTH (Lemma 1)
   Classical empirical CDF: variance → 0 as n → ∞
   Spatial CvM at fixed h: variance does NOT vanish
   → This is why we get a non-null limiting distribution

2. WEIGHTED CHI-SQUARE (Theorems 2-3)
   Unlike classical goodness-of-fit: T_n ≁ χ²(K-1)
   Instead: T_n ⟹ Σₘ λₘ χ²ₘ
   The weights λₘ depend on kernel K and domain geometry

3. COPULA-BASED MULTIVARIATE EXTENSION (Theorem 3)
   Generalizes univariate spatial CvM to multivariate
   Uses copulas to preserve marginal empiricals
   Eigenvalue structure inherited from Mercer decomposition

4. MIXING CONDITIONS (Throughout)
   α-mixing ensures covariance decay (Davydov inequality)
   Guarantees integrability and uniform convergence
   Critical for tightness and CLT-type results
